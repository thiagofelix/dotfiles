## dotfiles


1. Make sure homebrew is installed
2. Install brew bundle `brew bundle`
3. Link all dotfiles `stow . -t ~`
4. If you need unlink them `stow -D . -t ~`


### tools

* Alacritty: Terminal emulator
* Neovim: Text editor
* zsh/zinit: Shell

### macos

Set some macos defaults via `./.macos`
